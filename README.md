### Installation

1. Clone this repository
   ```sh
   git clone https://github.com/daianaadepaula/react-na-pratica-app.git
   ```
2. Go into the repository
   ```sh
   cd react-na-pratica-app
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the app
   ```sh
   npm run dev
   ```
